# KRDS-React TailwindCSS Plugin 예제

## 포함된 기능들

- [x] Typography
- [x] Color
- [x] Spacing
- [x] Radius

## 사용방법

### 0. 프로젝트 의존성 설치

```bash
pnpm install
```

### 1. Example 디렉토리 이동

```bash
cd examples/tailwindcss-plugin

```

### 2. 실행

```bash
pnpm example
```

### 3. 브라우저에서 확인

http://localhost:5173/ 에서 확인 가능
